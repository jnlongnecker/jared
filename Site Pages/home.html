What Is This Place?

This is my personal playground, a place for me to be creative and share my journey of CSS, HTML and Javascript as I work on projects that interest me. Everything that you see on this site will be 100% written in HTML, CSS and base Javascript with the p5 JS library. It will also all be written by me, meaning no fancy libraries other than p5.

My projects will be using the p5 canvas in order to visualize the simulation. The bouncing circles above are an example of one of these projects. Each project will come alongside a blurb detailing how I created it, the struggles that I had and what you can do with it, if it is interactable.

This site is developed and intented for desktop use, but does support mobile viewing. Since a lot of these p5 projects can be expensive in processing, any notable projects that are not optimized well enough for convenient mobile viewing will be noted.
